# House Price Prediction using Machine Learning

This project focuses on predicting house prices using machine learning techniques.  
The notebook includes data preprocessing, exploratory data analysis, feature handling, model training, and evaluation.

## Project Overview

Accurate house price prediction is an important real-world problem in the real estate industry.  
In this project, different machine learning workflows are applied to analyze housing data and build a predictive model capable of estimating property prices based on various features.

Because apparently humans enjoy turning square footage, bathrooms, and garage size into numerical destiny.

---

## Features

- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature selection and transformation
- Model training and evaluation
- House price prediction using regression models
